dotfiles
========

my dotfiles.

##Getting started
### 1. Install git, vim, zsh
If you are using CentOS

```
sudo yum update
yum install zsh
yum install vim 
yum install git
```

### 2. Clone this repository

```
git clone https://github.com/r-nmt000/dotfiles.git
```

### 3. Execute make command

```
make
```

**Done!!**


##ATTENTION

1. Make sure import eclipse_keybinds.epf manually.
